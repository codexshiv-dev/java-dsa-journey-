# Programming Handbook

> My personal Programming & DSA handbook for long-term learning.

## Purpose

This repository contains my structured notes on Programming Fundamentals, Java, DSA, Algorithms, Mathematics, and Interview Preparation.

The goal is not just to memorize solutions, but to understand concepts deeply and build strong problem-solving skills.

---

## Learning Principles

- Understand concepts deeply.
- Learn the "why" before the "how".
- Start with brute force.
- Optimize step by step.
- Practice consistently.
- Focus on patterns instead of memorizing code.
- Analyze Time & Space Complexity.
- Maintain clean and readable code.
